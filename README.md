# Supabase Minimal - Self-Hosted on Railway

Lightweight Supabase deployment for Railway with essential services:
- PostgreSQL Database
- Kong API Gateway
- Auth Service
- Edge Functions
- Storage
- Analytics
- Documentation

## Quick Start

1. Clone and deploy on Railway
2. Configure environment variables
3. Access Supabase at your Railway URL

For full documentation, see CONFIG.md in the original Supabase repo.
